Destacar texto

Aqui você vai aprender como destacar texto no JW Library.

Há duas maneiras de destacar um trecho do texto.

Toque e segure uma palavra para selecioná-la. Você pode arrastar as alças de seleção para definir o tamanho da sua marcação. Nas opções do menu que aparece, selecione a cor que você deseja usar para marcar aquele trecho.

Para destacar uma palavra ou frase em apenas um movimento, toque, segure e então arraste. Conforme você arrasta, o texto fica selecionado. Quando tiver terminado, um menu aparecerá brevemente. Você pode usar este menu para alterar a cor da marcação ou para apagá-la.

Para mudar a cor de uma marcação, toque no texto marcado e escolha uma nova cor. Para apagar uma marcação, toque no trecho marcado e toque em Apagar.
